Informasi Proyek

  • Basis: Website

  • Periferal yang digunakan: Html, Css Boostrap, PHP Laravel, JavaScript Jquery, Mysql
  • Deskripsi:
    Projek ini merupakan aplikasi website yang saya kembangkan selama bekerja di PT iGRA SOLUSI KREATIF, projek ini bermula memakai framework PHP Codeigniter 3 lalu saya di beri tanggug jawab untuk mengganti framework menjadi PHP Laravel.

    Projek ini bergerak dibidang ecommerce yang menjual berbagai produk dari berbagai produsen dan juga ada user mitra yang memiliki hak untuk menjual produk dari produsen yang dipilih, mitra juga mempunyai toko online dan domainnya sendiri
